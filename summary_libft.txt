ft_isalpha: Recibe un carácter c (como int) y retorna 1 si es alfabético o 0 si no lo es.
ft_isascii: Recibe un carácter c (como int) y retorna 1 si se encuentra en la tabla ASCII de 7 bits o 0 si no lo está.

ft_strlcpy: Copia el nº de caracteres indicado por size de un string src sobre un string dest, siendo el último carácter siempre un '\0'.
ft_strncmp: Compara los primeros n caracteres de dos strings s1 y s2 y devuelve el resto entre los valores ascii de la primera diferencia encontrada.
